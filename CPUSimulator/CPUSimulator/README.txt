This is a simple Console Application

This program runs 3 CPU schedualing algorithums
	1) FCFS - First Come First Serve
	2) SJF  - Shortest Job First
	3) MLFQ - Multi-Level Feedback Queue
and calculates the total time to execute, CPU utilization, waiting time, turnaround time, responce time, average waiting time, average turnaround time, and average responce time
