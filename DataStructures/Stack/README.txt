This is a simple Console Application

HOW TO USE:
This application required no input or direction from the user.

ABOUT THE PROGRAM:
This program implements a Stack class which
 - Consist of a pointer to local stack of ints
 - Manages dynamic memory (allocate and de-allocate memory)
 - Implements a default and copy constructor 
 - Implements a destructor,
 - Implements a push and pop method
 - Overload insertion (<<) operator as friend functions.
This program tests the Stack class by using its methods in Main
