This is a simple Console Application

HOW TO USE:
This application required no input or direction from the user.

ABOUT THE PROGRAM:
This program implements an Array class which
 - Manages dynamic memory (allocate and de-allocate memory)
 - Implements a default, explicit-value, and copy constructor 
 - Implements a destructor,
 - Implements a Sort and Search method.
 - Overload the assignment (=) and addition (+) as member functions, 
 - Overload insertion (<<) and the extraction (>>) operators as friend functions.
 - loads strings from a txt file to an Array
This program tests the Array class by using its methods in Main
