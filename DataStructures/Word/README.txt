This is a simple Console Application

HOW TO USE:
This application required no input or direction from the user.

ABOUT THE PROGRAM:
This program implements the abstract data type WORD (a String) using a linked list of characters which 
 - The state (private data members) contain a pointer to the front of the list of characters called "front".
 - Manages dynamic memory (allocate and de-allocate memory)
 - Implements a default, explicit-value, and copy constructor 
 - Implements a destructor,
 - Implements a Add, Insert, Search, Remove and Remove All method.
 - Overload the assignment (=) for both WORDs and strings, 
 - Overload insertion (<<) operators as friend functions.
This program tests the WORD class by using its methods in Main
