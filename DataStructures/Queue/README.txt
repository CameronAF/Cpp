This is a simple Console Application

HOW TO USE:
This application required no input or direction from the user.

ABOUT THE PROGRAM:
This program implements a Node class which
 - Consist of an int and a pointer to the next node and a pointer to the previous node
This program implements a doubly linked circular queue of integers.
 - Consist of pointer to the first Node
 - Manages dynamic memory (allocate and de-allocate memory)
 - Implements a default and copy constructor 
 - Implements a destructor
 - Implements an Enqueue and Dequeue
This program tests the Queue class by using its methods in Main
