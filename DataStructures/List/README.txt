This is a simple Console Application

HOW TO USE:
This application required no input or direction from the user.

ABOUT THE PROGRAM:
This program implements a Node class which
 - Consist of a string and a pointer to the next node
This program implements a Linked List class which
 - Consist of pointer to the head Node and the tail node
 - Manages dynamic memory (allocate and de-allocate memory)
 - Implements a default, explicit-value, and copy constructor 
 - Implements a destructor,
 - Implements a push back, push front, and pop front method
 - Overload insertion (<<) operator as friend functions.
This program tests the List class by using its methods in Main
