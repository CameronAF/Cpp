This Solution contains multiple project that implement different Data Structures.
Most of the Project implemented require no user input and are made for demonstration and academic purposes.

This solution Contains the following projects:
 1.	Array  - Implements an Array 
 2.	List   - Implements a Linked List 
 3.	List2  - Implements a Doubly-Linked List
 4.	Queue  - Implements a Doubly-Linked circular Queue
 5.	Stack  - Implements a Stack
 6.	Stack2 - Using Stacks, converts a user inputted postfix expression to infix expression.
 7.	Vector - Implements a Vector
 8.	Word   - Implements a Word (string) using a linked list of chars
