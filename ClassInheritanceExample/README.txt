This is a simple Console Application

HOW TO USE:
This application required no input or direction from the user.

ABOUT THE PROGRAM:
This program implements a parent class Duck and 4 child classes that inherit from Duck (Mallard, RubberDuck, DecoyDuck, and MotorizedDuck) which 
 - Implements a constructor
 - Implements a destructor
 - Implements a swim, fly, and quack method
This program tests the parent and children classes by using its methods in Main
